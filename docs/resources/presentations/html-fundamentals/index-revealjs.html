  • 📄 DOCTYPE = Building permit
  • 🏠 html = The entire house
  • 🧠 head = The attic (info not visible)
  • 🏡 body = The living space (what visitors see)

Head vs Body: What Goes Where?

👤 HEAD Section

Information ABOUT the page

  • Page title (browser tab)
  • CSS links
  • Meta information
  • Search engine data

Visitors don't see this directly

🏡 BODY Section

Content visitors see

  • Text and headings
  • Images and videos
  • Links and buttons
  • Everything visible

This is your actual content

Adding Images

🔗 Important Image Attributes:

  • src = where the image file is
  • alt = describes image for screen readers
  • width = controls width (optional)
  • height = controls height (optional)

Why alt text matters: Screen readers use it to describe images to visually impaired users. Always include meaningful descriptions!

Organizing Your Website Files


my-website/
├── index.html
├── about.html
├── css/
│   └── style.css
├── images/
│   ├── logo.png
│   └── photo.jpg
└── js/
    └── script.js

📝 Best Practices:

  • index.html = your homepage
  • Lowercase names with no spaces
  • Organize by type (css/, images/, js/)
  • Descriptive names (about.html not page2.html)
