HTML Fundamentals

The Foundation of Every Website

The HTML Structure

Think of HTML like a house

  • DOCTYPE = Building permit (declares rules)
  • html = The entire property line
  • head = The attic (wiring, insulation, not seen by guests)
  • body = The living rooms (decoration, furniture, seen by guests)
<!-- This tells the browser to use modern HTML5 -->
<!DOCTYPE html>

<!-- The root element -->
<html lang="en">

    <!-- Meta info (The Attic) -->
    <head>
        <title>My First Website</title>
    </head>

    <!-- Visible Content (The Living Room) -->
    <body>
        <h1>Welcome to my site!</h1>
    </body>

</html>

Head vs Body

👤 HEAD Section

Information ABOUT the page

  • Page title (browser tab)
  • Links to CSS files
  • Search engine keywords

Visitors don't read this directly.

🏡 BODY Section

Content visitors SEE

  • Headings & Paragraphs
  • Images & Videos
  • Links & Buttons

This is your actual content.

Essential Text Elements

These are the tags you will use 90% of the time.

<h1> to <h6> Headings (h1 is main title)
<p> Paragraphs of text
<strong> Bold (Important)
<em> Italic (Emphasized) 
<h1>My Portfolio</h1>
<h2>About Me</h2>
<p>I am a <strong>web developer</strong> based in Texas.</p>

Lists

Unordered List (Bullets)

<ul>
    <li>Coffee</li>
    <li>Tea</li>
</ul>

Ordered List (Numbers)

<ol>
    <li>Wake up</li>
    <li>Code</li>
</ol>

Links (Anchors)

Links connect pages together. They use the href attribute.

<!-- External Link -->
<a href="https://google.com">Visit Google</a>

<!-- Internal Link (Local File) -->
<a href="about.html">About Me</a>
Tip: Never use "Click Here" as link text. Use descriptive text (e.g., "Read my biography") for accessibility.

Images

Images are self-closing tags (no closing tag needed).

<img src="images/profile.jpg" alt="Portrait of Jane Smith">

Key Attributes

  • src (Source): Where the file is located.
  • alt (Alternative Text): Description for screen readers (Crucial for accessibility).

Attributes Explained

Attributes provide extra info inside the opening tag.

id="name" Unique identifier for ONE element
class="name" Group name for styling MANY elements

File Organization

Keep your project tidy to avoid broken links.

project-folder/
├── index.html          (Homepage - Must be named this!)
├── about.html
├── styles.css
└── images/             (Store all images here)
    ├── logo.png
    └── photo.jpg
Rule: Always use lowercase filenames with no spaces.
✅ my-file.html
❌ My File.html
