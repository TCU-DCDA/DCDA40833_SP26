# DCDA 40833: Senior Capstone (Spring 2026)

Welcome to the course repository for the DCDA Senior Capstone. This repository contains all necessary materials, lab instructions, and project guidelines for the semester.

## Course Overview
This capstone course is the culminating experience for the Digital Culture and Data Analytics (DCDA) major and minor. Students will undertake a significant "deep dive" project that synthesizes the skills and knowledge acquired throughout the program.

## Repository Structure

*   **`labs/`**: Contains weekly lab exercises and instructions (`lab01.md` - `lab06.md`).
*   **`project/`**: Contains detailed prompts and guidelines for the Semester Project.
    *   `project_prompt.md`: The main project proposal prompt.
    *   `project_prompt_alt.md`: An alternative version with additional clarifications for creative projects.

## Semester Project

The core of this course is the Semester Project, where students choose between two tracks:

1.  **Option 1 (Data Analysis):** A deep dive into Python-driven data analysis and visualization.
2.  **Option 2 (Digital Culture):** A deep dive into web development or multimodal authoring focused on a research topic or creative endeavor.

Please refer to the `project/` folder for full details on the proposal, research methodology, and deliverables.
